# Tourane News

Tourane News is a React/Vite authenticated newsroom app for reading reports, voting on posts, discussing stories, saving highlights, and managing approved-user access.

The app lives in:

```text
frontend/
```

Run locally:

```bash
cd frontend
npm install
npm run dev
```

Build:

```bash
cd frontend
npm run build
```

